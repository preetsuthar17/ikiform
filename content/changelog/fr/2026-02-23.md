### Ajouts
- Mode curseur a plage pour les champs de formulaire avec comportement de glissement ameliore.
- Meilleure gestion du curseur dans l'apercu de formulaire, le rendu simple et multi-etapes.
- Harnais de regression UI et couverture d'interaction pour les verifications de stabilite.
- Flux d'import/export de formulaires securise avec dialogues d'import et d'export soignes.
- Routage multilingue avec prefixe de langue (`/en/*`, `/es/*`) sur les pages publiques.
- Couverture de traduction pour l'accueil, le legal, le changelog, le tableau de bord, le constructeur de formulaires, l'analytique et les surfaces du constructeur IA.
- Moteur de metadonnees SEO centralise avec canonical localise, hreflang (`en`, `es`, `x-default`), OG, Twitter et gestion des robots.
- Routes de metadonnees App Router pour `sitemap.xml` et `robots.txt` localises.
- Arborescences de contenu localisees pour les entrees legales et de changelog.

### Modifications
- Migration de l'interface de base finalisee avec renforcement de production sur les primitives UI partagees.
- Analytique de formulaire et support de configuration des champs ameliores pour les valeurs liees au curseur.
- Guide de configuration README et documentation des routes etendus.
- Experiences de gestion et parametres de formulaires mises a jour pour supporter les flux de transfert securises.
- Generation statique cote serveur pour les pages marketing par langue afin d'ameliorer la performance et l'indexation.
- Comportement de proxy sensible a la langue pour les redirections et la normalisation de navigation.

### Corrections
- Problemes de comportement UI restaures identifies lors des verifications react-doctor.
- Definitions de style de variante menu-bar corrigees.
- Migration d'import de `framer-motion` vers `motion/react` completee dans les composants restants.

### Suppressions
- Composants descope/fonctionnalites mortes et modules UI obsoletes inutilises.
- Exports morts et surfaces d'aide heritees plus utilisees par l'application.
- Dependance `framer-motion` inutilisee du projet.
