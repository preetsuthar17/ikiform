### Добавлено
- Режим слайдера диапазона для полей формы с улучшенным поведением перетаскивания.
- Улучшена обработка слайдера в предварительном просмотре формы, одношаговом и многошаговом рендеринге.
- Тест-жгут регрессии UI и покрытие дымовых взаимодействий для проверки стабильности.
- Безопасный поток импорта/экспорта форм с отполированными диалогами импорта и экспорта.
- Многоязычная маршрутизация с префиксом локали (`/en/*`, `/es/*`) на публичных страницах.
- Покрытие перевода для главной страницы, юридических, журнала изменений, панели управления, конструктора форм, аналитики и поверхностей конструктора ИИ.
- Централизованный движок SEO-метаданных с локализованной обработкой canonical, hreflang (`en`, `es`, `x-default`), OG, Twitter и robots.
- Маршруты метаданных App Router для локализованных `sitemap.xml` и `robots.txt`.
- Локализованные деревья контента для юридических и журнальных записей.

### Изменено
- Завершена миграция базового UI с производственным усилением общих UI-примитивов.
- Улучшена поддержка аналитики форм и настройки полей для значений, связанных со слайдером.
- Расширено руководство по настройке README и документация маршрутов.
- Обновлены возможности управления формами и настроек для поддержки безопасных рабочих процессов передачи.
- Статическая серверная генерация для маркетинговых страниц локали для улучшения производительности и сканируемости.
- Поведение прокси с учетом локали для перенаправлений и нормализации навигации.

### Исправлено
- Восстановлены проблемы с поведением UI, выявленные во время проверок react-doctor.
- Исправлены определения стиля вариантов строки меню.
- Завершена миграция импорта с `framer-motion` на `motion/react` в оставшихся компонентах.

### Удалено
- Неиспользуемые компоненты функций descope/мертвых и устаревшие модули UI.
- Мертвые экспорты и устаревшие вспомогательные поверхности, которые больше не используются приложением.
- Неиспользуемая зависимость `framer-motion` из проекта.
