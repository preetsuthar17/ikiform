### 新增
- 具有改进拖动行为的表单字段范围滑块模式。
- 改进了表单预览、单步骤和多步骤渲染中的滑块处理。
- 用于稳定性检查的 UI 回归工具和烟雾交互覆盖。
- 具有精细导入和导出对话框的安全表单导入/导出流程。
- 公共页面上带有区域设置前缀的多语言路由（`/en/*`、`/es/*`）。
- 主页、法律、更新日志、仪表板、表单构建器、分析和 AI 构建器界面的翻译覆盖。
- 具有本地化 canonical、hreflang（`en`、`es`、`x-default`）、OG、Twitter 和 robots 处理的集中 SEO 元数据引擎。
- 本地化 `sitemap.xml` 和 `robots.txt` 的 App Router 元数据路由。
- 法律和更新日志条目的本地化内容树。

### 变更
- 完成了具有生产加固的基础 UI 迁移，覆盖共享 UI 基元。
- 改进了滑块相关值的表单分析和字段配置支持。
- 扩展了 README 设置指导和路由文档。
- 更新了表单管理和设置体验，以支持安全传输工作流。
- 区域设置营销页面的静态服务端生成，以提高性能和可爬取性。
- 用于重定向和导航规范化的区域设置感知代理行为。

### 修复
- 恢复了在 react-doctor 检查期间发现的 UI 行为问题。
- 修正了菜单栏变体样式定义。
- 完成了剩余组件中从 `framer-motion` 到 `motion/react` 的导入迁移。

### 移除
- 未使用的 descope/死功能组件和过时的 UI 模块。
- 应用不再使用的死导出和旧版辅助界面。
- 从项目中移除未使用的 `framer-motion` 依赖项。
