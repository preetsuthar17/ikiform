### जोड़ा गया
- प्रोग्रामेटिक एक्सेस के लिए फ़ॉर्म API कुंजियाँ
- API कुंजी प्रबंधन इंटरफ़ेस
- सुरक्षित API प्रमाणीकरण
- API दस्तावेज़ीकरण और एंडपॉइंट
- प्रोग्रामेटिक फ़ॉर्म सबमिशन समर्थन

### परिवर्तित
- API कुंजियों के साथ उन्नत फ़ॉर्म सुरक्षा
- API एक्सेस प्रबंधन में सुधार
