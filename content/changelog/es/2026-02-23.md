### Agregado
- Modo de control deslizante de rango para campos de formulario con comportamiento de arrastre mejorado.
- Mejor manejo del control deslizante en la vista previa del formulario, renderizado de un solo paso y de varios pasos.
- Arnés de regresión de la interfaz de usuario y cobertura de interacción de humo para controles de estabilidad.
- Flujo seguro de importación/exportación de formularios con diálogos de importación y exportación pulidos.
- Enrutamiento multilingüe con prefijo de locale (`/en/*`, `/es/*`) en páginas públicas.
- Cobertura de traducciones para superficies de inicio, legal, changelog, panel, constructor de formularios, analíticas y AI builder.
- Motor centralizado de metadatos SEO con canonical localizado, hreflang (`en`, `es`, `x-default`), OG, Twitter y directivas robots.
- Rutas de metadatos del App Router para `sitemap.xml` y `robots.txt` localizados.
- Árboles de contenido localizados para documentos legales y entradas del changelog.

### Cambios
- Se finalizó la migración de la interfaz de usuario base con refuerzo de producción en primitivas de interfaz de usuario compartidas.
- Análisis de formularios mejorado y soporte de configuración de campos para valores relacionados con el control deslizante.
- Guía de configuración README ampliada y documentación de ruta.
- Experiencias de configuración y administración de formularios actualizadas para admitir flujos de trabajo de transferencia seguros.
- Generación estática del lado del servidor para páginas de marketing con locale, mejorando rendimiento y rastreabilidad.
- Comportamiento del proxy con conocimiento de locale para redirecciones y normalización de navegación.

### Corregido
- Se restauraron los problemas de comportamiento de la interfaz de usuario identificados durante las comprobaciones de reaccionar-doctor.
- Se corrigieron las definiciones de estilo de las variantes de la barra de menú.
- Se completó la migración de importación de `framer-motion` a `motion/react` en los componentes restantes.

### Eliminado
- Componentes de funciones inactivas/descope no utilizados y módulos de interfaz de usuario obsoletos.
- Exportaciones inactivas y superficies auxiliares heredadas que la aplicación ya no utiliza.
- Dependencia de `framer-motion` no utilizada del proyecto.
